Temporary Hotfix for DZAI with Epoch 1.0.2.4a
---

About:

This hotfix is meant to address the problem of certain player skin(s) being friendly with AI sides.


Install instructions:

Copy DZAI folder over your existing DZAI installation and overwrite files if prompted.


Uninstallation instructions:

When Epoch is updated to 1.0.5, redownload DZAI and reinstall without applying any hotfix files.
